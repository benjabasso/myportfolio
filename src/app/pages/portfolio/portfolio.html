Medical System

Proyecto personal de aplicación web para gestión de pacientes clínicos.

Repositorio Demo

Información del Proyecto

Este proyecto es un sistema de gestión clínica full-stack desarrollado con Node.js, Express, PostgreSQL y React, diseñado para administrar pacientes, usuarios y operaciones internas de manera segura y eficiente.

Cuenta con autenticación mediante JWT, manejo centralizado de errores, formularios validados, componentes reutilizables y un panel de control moderno.

Incluye buenas prácticas de desarrollo, control de versiones con Git, arquitectura escalable y principios sólidos de ingeniería de software. Próximamente incorporará un sistema de roles y consultas médicas.

Human Art Casting

Desarrollo de JSON y API para formulario de inscripción.

Demo

Información del Proyecto

Desarrolle una API en PHP para Wordpress utilizando el plugin Code Snippets.

El cliente pidió la conexión de un formulario de inscripción con su CRM (JetPack CRM), para no tener gastos en mantenimiento ni subscripciones a plugins premium de Forms o Webhooks, decidí implementar un Snippet desarrollando una API conectada con las credenciales del CRM y un formulario desarrollado en Contact Form 7.

Incluyendo carga de archivos (fotos para casting) y visualizacion de las mismas en una ruta de la página, checkbox para selecciones multiples y mailing automático.
